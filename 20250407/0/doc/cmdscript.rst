cmdscript Module
===============

This module provides a command-line interpreter with two commands:

- ``bless <string>``: Prints "Be blessed, <string>!"
- ``sendto <string>``: Prints "Go to <string>!"

Usage
-----

Run interactively:

.. code-block:: bash

   python3 cmdscript.py

Run with a file input:

.. code-block:: bash

   python3 cmdscript.py input.txt
